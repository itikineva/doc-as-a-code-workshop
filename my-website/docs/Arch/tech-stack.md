## Технологический стек HeroTask

| Компонент | Технология | Версия | Назначение |
| --------- | ---------- | ------ | ---------- |
| Backend | Java / Spring Boot | 17 / 3.x | Основная бизнес-логика, REST API |
| База данных | PostgreSQL | 15 | Хранение данных о героях, инцидентах и задачах |
| Кэш | Redis | 7 | Кэширование статусов, хранение сессий |
| Очередь событий | Apache Kafka | 3.x | Асинхронная доставка событий между сервисами |
| Frontend | React | 18 | Веб-интерфейс диспетчера |
| Хостинг | Yandex Cloud | — | Вычислительные ресурсы и управляемые сервисы |
| Хранилище файлов | Yandex Cloud S3 | — | Фотографии профилей, вложения |
| Логирование | ElasticSearch + Kibana | 8.x | Сбор и визуализация логов |
| Мониторинг | Prometheus + Grafana | — | Метрики производительности и алерты |
| CI/CD | GitLab CI | — | Сборка, тестирование и деплой |

:::note
Версии зафиксированы в `pom.xml` и обновляются только после прохождения полного регрессионного тестирования.
:::
